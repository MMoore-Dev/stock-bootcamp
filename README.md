## Purpose
This repository documents my journey in developing stock market analysis tools using Python.  
The goal is to build skills in data analysis and automation.

## Current Features
- Profit/Loss calculator
- Percentage return calculation
- Trade decision logic (profit, loss, break-even)
- Multi-trade analysis using loops
- Reusable analysis functions (modular design)
- CSV file input for stock price data
- Basic price analysis (average, min, max) 
- Data analysis using pandas
- Derived metrics (daily change, percent change)

## Tech Stack
- Python
- Visual Studio Code
- Git & GitHub
